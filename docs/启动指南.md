# A股智能选股 + 趋势预测系统 — 启动指南

## 环境要求

| 工具 | 版本 |
|------|------|
| Python | 3.11.9 (工程虚拟环境) |
| Node.js | ≥ 18.0 |
| Redis (Memurai) | ≥ 6.2 (本机已安装) |

---

## 一、后端启动

### 1. 激活虚拟环境 & 安装依赖

```bash
cd backend
py -3.11 -m venv .venv
# 安装 TA-Lib (Windows 需单独 wheel)
.venv/Scripts/pip install https://github.com/cgohlke/talib-build/releases/download/v0.4.28/TA_Lib-0.4.28-cp311-cp311-win_amd64.whl
# 安装其余依赖
.venv/Scripts/pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple
```

### 2. 配置环境变量

```bash
cp .env.example .env
# 编辑 .env, 至少确认:
# - REDIS_PASSWORD= (按实际密码填写)
# - LLM_CONFIG_SECRET= (首次启动自动生成)
```

### 3. 启动服务

```bash
cd backend
.venv/Scripts/uvicorn app.main:app --reload --host 127.0.0.1 --port 8000
```

### 4. 验证

```bash
curl http://127.0.0.1:8000/api/v1/health
# {"code":0,"msg":"ok","data":{"app":"gupiaoagent","status":"up"},"disclaimer":"..."}
```

---

## 二、前端启动

```bash
cd frontend
npm install
npm run dev
# 默认 http://localhost:5173, 代理 /api -> 127.0.0.1:8000
```

---

## 三、初次使用

1. **配置大模型**：http://localhost:5173/settings/llm
   - 选择 Provider, 填入 URL+Key, 获取模型列表, 保存+测试
2. **选股**：首页勾选策略 → 开始选股
3. **个股分析**：点击股票代码 → K 线+预测+风控
4. **回测**：/backtest → 输入区间 → 运行

---

## 四、常见问题

- Redis 连不上: 检查密码 `.env` `REDIS_PASSWORD`
- TA-Lib 失败: Python 3.11 需下载 cgohlke wheel
- akshare 超时: 网络波动, 已内置 3 次重试
