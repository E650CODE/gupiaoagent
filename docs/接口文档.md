# A股智能选股系统 — 接口文档

基础路径: `http://127.0.0.1:8000/api/v1`  
OpenAPI 文档: `http://127.0.0.1:8000/docs`

---

## 通用响应格式

```json
{
  "code": 0,           // 0=成功, >0=业务错误
  "msg": "ok",         // 提示信息
  "trace_id": "hex12", // 链路追踪ID
  "data": {},          // 业务数据
  "disclaimer": "..."  // 合规风险提示
}
```

错误码：
| code | 含义 |
|------|------|
| 0 | 成功 |
| 4400 | 参数错误 |
| 2001 | 数据源异常 |
| 3001 | LLM 配置错误 |
| 3002 | LLM 请求失败 |
| 5000 | 服务器内部错误 |

## 业务接口

### POST /stock/select — 多因子选股

请求：
```json
{
  "strategies": ["ma_bull", "macd_golden"],
  "top_n": 20,
  "enable_ai": true,
  "enable_risk": true,
  "universe": null,
  "params": null
}
```
响应 data 字段：
```json
{
  "strategies": ["ma_bull"],
  "rules": ["MA5 > MA10 > MA20 > MA60"],
  "stocks": [{"code":"600519","name":"贵州茅台","close":1402,"pct_chg":-0.86,"ma5":1407.81,"ma20":1401.59,"macd_dif":-2.78,"rsi6":43.2,"vol_ratio":1.5,"risk_level":"低"}],
  "count": 1,
  "ai_explanation": {"logic":"...","common_features":["..."],"sector_hot":"...","summary":"..."}
}
```

### POST /stock/predict — 趋势预测

请求 `{"code":"600519","horizon_days":5}`

响应 data：
```json
{
  "prediction": {
    "direction": "up|down|range",
    "prob": 0.78,
    "price_low": 1380,
    "price_high": 1420,
    "horizon_days": 5,
    "reasoning": "...",
    "risk_warning": "...",
    "confidence": "高"
  },
  "risk": {"risk_level":"低","risk_points":[],"advice":"..."}
}
```

### GET /stock/detail/{code} — 个股详情

数据: `kline[]` + `factors[]` + `funds[]`

### POST /backtest/run — 回测

请求 `{"strategy":"ma_bull","start":"20240101","end":"20240601","params":{"initial_cash":100000,"hold_days":5}}`

响应 data：参见回测看板。

## LLM 配置接口

| 方法 | 路径 | 说明 |
|------|------|------|
| GET  | /llm/config | 获取配置 (api_key 掩码) |
| PUT  | /llm/config | 更新配置 `{"agent":"selector","config":{...}}` |
| POST | /llm/models | 拉模型列表 `{"provider":"deepseek","base_url":"...","api_key":"..."}` |
| POST | /llm/test   | 连通性测试 |
